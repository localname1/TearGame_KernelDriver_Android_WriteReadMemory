## 📋 简介

本项目是一个经过完整优化的 Android GKI（Generic Kernel Image）内核驱动模块。由泪心独立开发并优化，解决了原版本的所有已知问题，包括加载失败、内核崩溃、兼容性等问题。

### ✨ 特性

- ✅ 完整解决加载重启和 kernel panic 问题
- ✅ 支持驱动读写功能
- ✅ 支持驱动初始化
- ✅ 兼容内核版本 4.14.186、5.10、6.6、6.12
- ✅ 默认驱动节点：`/dev/TearGame`
- ✅ 驱动读写隐藏功能
- ✅ **完整绘制作者对接驱动源码，直接使用，永久无验证，无服务器密钥**
- ⚠️ 未实现 lsmod 隐藏（保持可见性）

## 🔨 编译说明

### 前置要求

在编译驱动模块之前，必须先编译内核源码。请根据您的设备下载对应的内核源码：

- **小米设备**: `android_kernel_xiaomi` (开源版本)
- **一加设备**: `oneplus_kernel` (开源版本)
- **LineageOS**: LineageOS 开源内核源码

### 编译步骤

1. 下载并编译对应的内核源码
2. 链接内核源码到驱动模块目录
3. 编译驱动模块（仅生成 `.ko` 文件）

```bash
# 示例编译命令
make -C /path/to/kernel/source M=$(pwd) modules
```

### 使用预编译驱动

非驱动开发者可以直接使用已编译好的驱动文件，自行对接使用。
